# ccl-portail-membre
Application PWA pour le Club Cycliste de Lévis (Volet Jeunesse). Solution "Headless" (Next.js/Supabase) connectée à Odoo. Centralise la logistique terrain (présences, calendrier), la communication (chat) et la pédagogie (académie vidéo). Gère nativement les familles multi-profils, la rotation des coachs et l'affichage des partenaires.
